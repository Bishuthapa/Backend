## **Setting up Backend**

First, create the `package.json` using the **`npm init`** command.


```json
{
  "name": "revbackend",
  "version": "1.0.0",
  "description": "backend revision",
  "keywords": [
    "JavaScript",
    "backend"
  ],
  "license": "ISC",
  "author": "Author Name",
  "type": "module",
  "main": "index.js",
  "scripts": {
    "start": "nodemon src/index.js", //use nodemon to run the server automatically
    "test": "echo \"Error: no test specified\" && exit 1"
  }
}

```

---

<br />

Install the `node_modules` using the **`npm install i`** command.

 Create .env file and add the following contents:

```json
{
  "PORT": 3000
}
```

Create a public folder for images and videos.
In the public folder add .gitignore file.

```json
node_modules
.env //add other files that you want to ignore
```

 Now Create a Folder directory 

> src
> - app.js
> - constant.js
> - index.js

--- 

<br />


Use nodemon to run the server. Install the dev dependencies.
>To install nodemon use **`npm install -D nodemon`** command.
